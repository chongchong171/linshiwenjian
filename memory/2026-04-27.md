## 🐑 羊了个羊深度分析

**背景：** 女王大人问我"IAA 小游戏"是什么，然后让我深度研究羊了个羊的成功要素。

**核心发现：**
- 羊了个羊团队：初始 3 人，开发约 3 个月，7 天 DAU 破亿，峰值 QPS 21 万
- 成功核心：**反常识设计 → 情绪爆发 → 社交传播 → 越骂越火**
- 8 个核心成功要素：极简玩法、断崖式难度、情绪操控、社交裂变、每日挑战、简单变现、随机性、通关奖励
- 第二关通关率 <0.1%，死局率 99.5%+

**关键洞察：**
- 不是因为它好玩，而是因为它"反常识"
- 渐进式难度 → 用户觉得正常不分享；断崖式难度 → 用户觉得不正常要吐槽
- 吐槽就是传播
- 赌徒黑洞循环：投入时间 → 失败 → 不甘心 → 继续投入 → 失败 → 更不甘心

## 🖥️ Cocos Creator 安装过程

**问题：** 女王大人的 Windows 11 电脑安装被阻止
- 原因：腾讯电脑管家 + Windows 智能应用控制
- 解决：先安装 Cocos Dashboard，然后通过 Dashboard 安装 Cocos Creator
- 最终：成功安装 Cocos Creator 3.8.8

**注意事项：**
- Cocos Dashboard 是启动器/管理器，不是开发工具
- 通过 Dashboard 安装 Cocos Creator 比直接运行 EXE 更可靠
- Windows 11 智能应用控制可能阻止安装，需要关闭或下载 ZIP 免安装版
