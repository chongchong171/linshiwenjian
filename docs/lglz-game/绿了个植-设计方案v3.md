## 二、核心玩法设计

### 1. 滑动相邻消除（核心机制）

**借鉴来源：** 三消查找算法 + 连线消除游戏

**玩法描述：**
- 玩家在花盆中滑动相邻的相同植物
- 滑动后直线连接，2+ 植物消除
- 消除后上方植物下落填补空位
- 连锁消除机制

**核心算法：**
```typescript
// 三消查找算法（适配绿了个植）
class PlantMatchFinder {
  // 递归查找相邻相同植物
  findMatches(startPlant: Plant, grid: Plant[][]): Plant[] {
    const matches: Plant[] = []
    const visited = new Set<string>()
    
    const dfs = (plant: Plant) => {
      const key = `${plant.layer}-${plant.row}-${plant.col}`
      if (visited.has(key)) return
      visited.add(key)
      
      matches.push(plant)
      
      // 查找相邻位置
      const directions = [
        { row: -1, col: 0 },
        { row: 1, col: 0 },
        { row: 0, col: -1 },
        { row: 0, col: 1 }
      ]
      
      for (const dir of directions) {
        const adjacent = grid[plant.row + dir.row]?.[plant.col + dir.col]
        if (adjacent && adjacent.type === plant.type && adjacent.visible) {
          dfs(adjacent)
        }
      }
    }
    
    dfs(startPlant)
    return matches
  }
}
```

### 2. 多层堆叠布局

**借鉴来源：** 羊了个羊堆叠牌拾取

**玩法描述：**
- 多层植物堆叠在花盆中
- 上层植物遮挡下层植物
- 消除上层后，下层植物可见
- 数据表格抽象管理状态

**实现方案：**
```typescript
class PlantStack {
  layers: Plant[][][]
  
  // 检查植物是否可滑动
  canSwipe(plant: Plant): boolean {
    if (!plant.visible) return false
    
    // 检查是否有相邻的相同类型植物
    const adjacent = this.getAdjacentPlants(plant)
    return adjacent.some(adj => adj.type === plant.type)
  }
  
  // 刷新所有植物状态
  refreshAllPlants() {
    for (const layer of this.layers) {
      for (const row of layer) {
        for (const plant of row) {
          if (!plant.visible && !this.isBlocked(plant)) {
            plant.visible = true
          }
        }
      }
    }
  }
}
```

## 三、道具系统设计

**借鉴来源：** 羊了个羊三种道具

### 1. 道具列表

| 道具 | 功能 | 获取方式 |
|------|------|---------|
| 放大镜 | 提示可消除的植物对 | 每日登录/分享 |
| 除草剂 | 消除指定位置的植物 | 关卡奖励/分享 |
| 洗牌 | 随机打乱所有植物位置 | 关卡奖励/分享 |
| 复活 | 游戏失败后继续 | 观看广告/分享 |

### 2. 实现方案

```typescript
class PowerUpManager {
  // 放大镜：提示可消除的植物对
  useMagnifier() {
    const matches = this.findAnyMatch()
    if (matches) {
      this.highlightPlants(matches)
      this.scheduleClearHighlight(3000)
    }
  }
  
  // 除草剂：消除指定位置的植物
  useWeedkiller(plant: Plant) {
    this.removePlant(plant)
    this.refreshAllPlants()
  }
  
  // 洗牌：随机打乱所有植物位置
  useShuffle() {
    const positions = this.getAllPlantPositions()
    const shuffled = this.shuffleArray(positions)
    this.setPlantPositions(shuffled)
    this.refreshAllPlants()
  }
  
  // 复活：游戏失败后继续
  useRevive() {
    this.restoreGame()
    this.refreshAllPlants()
  }
}
```

### 2. 实现方案

```typescript
class RankManager {
  // 上传分数
  async uploadScore(score: number) {
    await wx.cloud.callFunction({
      name: 'updateScore',
      data: { score }
    })
  }
  
  // 获取好友排行榜
  async getFriendRank() {
    const openDataContext = wx.getOpenDataContext()
    openDataContext.postMessage({ type: 'getRank' })
  }
  
  // 获取全国排行榜
  async getNationalRank(type: string) {
    const res = await wx.cloud.callFunction({
      name: 'getRankList',
      data: { type, limit: 100 }
    })
    return res.result
  }
}
```
